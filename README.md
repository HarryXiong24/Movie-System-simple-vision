# Vue-ts-rem-start

## Description

Vue-cli4, Typescript startup template, use rem adaptation.

## Project setup

``` bash
yarn install
```

### Compiles and hot-reloads for development

``` bash
yarn serve
```

### Compiles and minifies for production

``` bash
yarn build
```

### Lints and fixes files

``` bash
yarn lint
```
